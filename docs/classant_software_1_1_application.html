Detailed Description

Wrapper class which indirectly exposes the Unity Application to the Game API.

All public members of this class are exposed to Lua. Inherits from uLua.ExposedMonoBehaviour.

◆ RequestedResolution

string antSoftware.Application.RequestedResolution
get

Returns the requested resolution based on the current WindowMode setting.

FullScreenWindow returns NativeResolution. ExclusiveFullScreen returns FullScreenResolution. Windowed returns WindowedResolution. The resolution is returned as a string with the format: {Width}x{Height}.
